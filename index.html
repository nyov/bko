Using PXE-Knife over HTTP

Following are the gpxe images which are imbedded with script to access PXE-Knife over http on this site

The embedded scripts assume that you have internet connectivity based on DHCP

We will have custom pxe images for static IP addresses soon!!

gpxe image for qemu or floppy qemu example 

qemu -fda gpxe.dsk

gpxe image for CD-ROM qemu example 

qemu -cdrom gpxe.iso

gpxe image for USB

You can also try out PXE-Knife over HTTP with your existing pxe, following is the example of using standard pxe with qemu for booting with PXE-Knife over HTTP

qemu -fda gpxe.dsk -net nic -net user -bootp http://alien.doesntexist.org/pxeknife/pxelinux.0

Refer Etherboot wiki for more details
