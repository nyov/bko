How to replicate the BKO setup on your machine?

  1. Get the BKO.git and gpxe repositories. You can use the script setup.sh for this work.
  2. Edit the BKO/configure to suit your needs. It can be done with any text editor 
    vi BKO/configure
    Make sure that you provide proper values for atleast GPXE_GIT_LOCATION, BOOT_URL and ISO_LOCATION_LOCAL Other entries are optional, you can leave them untouched. Please refer bellow for more details on the meaning of the configuration variables.
  3. Follow the following instructions 
    cd BKO/install_help
./deploy.sh
./download_ISO.sh

Details about configuration parameters

  1. GPXE_GIT_LOCATION: path to the place where gpxe git is placed. If you have used the setup.sh script, then this path should be ../gpxe. the gpxe folder can be moved anywhere as long as the path is given properly.
  2. BOOT_URL: gpxe will contact this URL for initial boot. It should point to the BKO folder which is created by setup.sh script. and it must be HTTP URL and not linux path.
